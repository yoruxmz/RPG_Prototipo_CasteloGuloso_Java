Esse nada mais é do que um simples joguinho de RPG em forma de texto que eu estava desenvolvendo enquanto treinava java, acabei gostando do resultado e decidi lançar aqui.

Testem a vontade e estou aberto para receber opiniões sobre!
## 🚀 Como rodar o jogo
1. Tenha o Java JDK instalado.
2. Baixe o arquivo `Main.java`.
3. No terminal, use:
   javac Main.java
   java Main
